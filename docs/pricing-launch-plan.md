# 블링빌 요금제 출시 실행 계획

기준 문서: `블링빌_요금제_수익성_정리_최종수정.docx` (2026-07-23)

## 확정 정책

- 무료 + 크레딧: 고객관리·견적 작성 무료, 발행·발송 시 차감
- 입문: 월 9,900원, 세금계산서 10건, 원장·미수·매입 관리
- Pro: 월 29,000원/50건 또는 49,000원/100건. 기능은 동일
- 초과 사용은 자동 과금하지 않고 사용자가 크레딧을 구매
- 신규 가입 3크레딧은 사업자등록번호당 한 번만 지급

## 결제 연동 전에 결정할 것

1. PortOne에서 사용할 실제 PG 채널과 카드 정기결제 계약
2. 가격의 부가세 포함 여부와 현금영수증/세금계산서 처리 방식
3. 결제 실패 재시도 횟수, 해지 효력일, 미사용 크레딧 유효기간·환불
4. 플랜 업그레이드/다운그레이드 적용일과 일할 계산
5. 기존 가입자의 무료 3크레딧 지급 기준일

정책이 없는 상태에서 금전 결과를 임의로 만들지 않기 위해 현재 유료 플랜 간 변경과 자동 환불·해지는 잠가 두었다.

## 출시 검증

- 결제 webhook은 중복 수신되어도 한 번만 적립
- 발행 실패 시 차감 롤백, 성공 응답과 원장 기록이 일치
- Pro 50/100의 기능 권한은 동일하고 포함 건수만 다름
- 월 포함 건수를 모두 쓰면 자동 결제가 아닌 크레딧 구매 안내
- 사업자번호 변경으로 무료 크레딧을 중복 수령할 수 없음
- 최초 구독 결제와 갱신 주문을 구분하며, 최초 결제 실패가 기존 플랜을 미납으로 바꾸지 않음
- 1월 31일 결제도 2월 말일로 예약되어 다음 달을 건너뛰지 않음
